# Roadmap - Ministrify

## Быстрі перемоги (Quick Wins)

### Вхід по магічному посиланню
- [ ] Відправити email з посиланням замість паролю
- [ ] Люди постійно забувають паролі - це вирішує проблему
- [ ] Посилання дійсне 15 хвилин

### Google/Apple Sign-In
- [ ] 80% користувачів віддають перевагу соціальному входу
- [ ] Laravel Socialite для реалізації
- [ ] Прив'язка до існуючого акаунту по email

### "Запам'ятати пристрій"
- [ ] Галочка "Довірений пристрій" при вході
- [ ] Сесія 30 днів замість 12 годин
- [ ] Список довірених пристроїв в налаштуваннях

---

## UX покращення

### QR-код для входу
- [ ] Показати QR на екрані комп'ютера
- [ ] Відсканувати телефоном → залогінений (як Telegram Web)
- [ ] Корисно для швидкого входу на чужому пристрої

### Темна тема по розкладу
- [ ] Автоматично вмикати ввечері (після 20:00)
- [ ] Налаштування в профілі

### Offline режим (PWA)
- [ ] Кешування розкладу служінь
- [ ] Перегляд без інтернету
- [ ] Синхронізація при відновленні з'єднання

---

## Для росту

### Реферальна програма
- [ ] "Запроси іншу церкву - отримай місяць безкоштовно"
- [ ] Унікальне реферальне посилання для кожної церкви
- [ ] Дашборд з статистикою запрошень

### SEO для публічних сторінок
- [ ] Покращити `/c/{slug}` сторінки
- [ ] Open Graph мета-теги
- [ ] Структуровані дані (JSON-LD)
- [ ] Sitemap для пошукових систем

### Мобільний застосунок
- [ ] PWA обгортка для App Store / Google Play
- [ ] Push notifications через Firebase
- [ ] Capacitor або PWA Builder

## Автоматизація

### AI помічник для планування
- [ ] "Кого поставити на неділю?"
- [ ] Аналіз доступності волонтерів
- [ ] Врахування ротації та вподобань
- [ ] Автоматичні пропозиції складу

### Розумні нагадування
- [ ] Контекстні нагадування на основі позиції
- [ ] "Завтра служіння, не забудь гітару" (для музикантів)
- [ ] "Завтра служіння, приходь о 9:00" (для sound-інженерів)
- [ ] Інтеграція з календарем (Google Calendar, Apple Calendar)
